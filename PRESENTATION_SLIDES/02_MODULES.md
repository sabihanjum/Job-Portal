# SLIDE 2: LIST OF MODULES

## System Modules Overview

---

## 1. Authentication & User Management Module
**Features:**
- Multi-role registration (Candidate, Recruiter, Admin)
- JWT-based secure authentication
- Role-based access control (RBAC)
- Profile management
- Password encryption

---

## 2. Resume Management Module
**Features:**
- Secure document upload (PDF, DOCX, Images)
- AI-powered resume parsing using NLP
- Automatic extraction of:
  - Personal information
  - Skills and technologies
  - Work experience
  - Education details
- Resume version control

---

## 3. Job Management Module
**Features:**
- Job posting creation and editing
- Job listing with advanced filters
- Salary range and experience level
- Required skills specification
- Job status management (Active/Inactive)

---

## 4. AI Matching Engine Module
**Core Technology:**
- Semantic similarity using SentenceTransformers
- Match percentage calculation
- Skill gap analysis
- Explainability heatmap

**Outputs:**
- Match score (0-100%)
- Matched skills
- Missing skills
- Personalized recommendations
